# Features
🖊️ Smooth handwriting and drawing
🎨 Change pen color and size
🧽 Eraser tool for corrections
📄 Add multiple pages
↩️ Undo & Redo functionality
📄 Export drawings as PDF
📂 Open and share saved PDFs
👆 Single-touch drawing & multi-touch scroll support

#### Tech Stack ####
Kotlin , XML , Android Canvas API , Open AI

#### Futures Updates ####
1.AI-Powered PDF Summarization - Automatically generate short and easy-to-understand summaries from handwritten or imported PDFs to help users revise quickly.
2.Smart Study Notes Generator - Convert long notes into key points, highlights, and bullet summaries for faster learning.
3.Text-to-Speech (Audio Notes) - Convert notes or summaries into audio so users can listen and learn.
4.Handwriting Recognition (OCR) - Convert handwritten notes into editable digital text.
5.Cloud Sync & Backup (Firebase) - Save notes online and access from multiple devices.
